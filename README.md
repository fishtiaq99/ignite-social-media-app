# Ignite - Campus Social Media App

## Project Overview
Ignite is a university-focused social media platform designed for students to connect, share content, and engage with campus activities. The platform encourages meaningful interaction within the campus while ensuring proper moderation and administrative control.

## Key Features
- User Registration & Login
- Follow / Unfollow Users 
- Create, Like, and Comment on Posts
- Trending Posts Algorithm
- Home Feed 
- Advanced Search (Users, Hashtags, Posts)
- Content Reporting System
- Admin Dashboard for Moderation
- System-Wide Alert Broadcasting

## Architecture & Design
- Strategy Pattern for Search & Sorting
- Repository Pattern for Data Access
- Object-Oriented Design Principles
